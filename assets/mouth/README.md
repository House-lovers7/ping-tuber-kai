# 口形状アセット仕様

## 必要な画像ファイル

以下の7つのPNG画像が必要です：

| ファイル名 | 母音 | 口形状 | 説明 |
|-----------|------|--------|------|
| `a.png` | あ | 大きく開く | 縦に大きく開いた口 |
| `i.png` | い | 横に広げる | 横に広げた口（笑顔のような形） |
| `u.png` | う | すぼめる | 唇をすぼめた口 |
| `e.png` | え | 少し開く+横 | 軽く開いて横に広げた口 |
| `o.png` | お | 丸く開く | 丸く開いた口 |
| `n.png` | ん | 軽く閉じ | 軽く閉じた口（ハミング） |
| `closed.png` | - | 閉じ | 完全に閉じた口 |

## 画像仕様

- **形式**: PNG（透過推奨）
- **推奨サイズ**: 400x400 ピクセル（設定で変更可能）
- **背景**: 透過またはクロマキー用の単色

## 使用タイミング

| 状況 | 表示画像 |
|------|----------|
| 有声母音 a | `a.png` |
| 有声母音 i | `i.png` |
| 有声母音 u | `u.png` |
| 有声母音 e | `e.png` |
| 有声母音 o | `o.png` |
| 撥音 N（ん） | `n.png` |
| 無声母音（A,I,U,E,O） | `closed.png` |
| 子音のみ | `closed.png` |
| ポーズ・無音 | `closed.png` |

## テスト用プレースホルダー生成

テスト用のプレースホルダー画像を生成するには：

```bash
uv run python scripts/generate_placeholder_images.py
```

これにより、各口形状を簡易的に表現したプレースホルダー画像が生成されます。

## カスタム画像の作成

1. 上記7つのPNGファイルを作成
2. このディレクトリに配置
3. または `--assets` オプションで別ディレクトリを指定

```bash
uv run ping-tuber --text "こんにちは" --assets ./my_custom_mouth/
```
